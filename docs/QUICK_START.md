## 🚀 Quick Start (5 Minutes)

### Step 1: Install (1 min)

1. **Open** the `.dmg` file
2. **Drag** app to Applications folder
3. **Launch** the app

**First time?** Right-click → Open (bypasses Gatekeeper)

---

### Step 2: Configure AI (2 min)

**You need an AI provider to generate resumes.**

#### Option A: OpenAI (Cloud)

1. **Get API Key:**
   - Go to: https://platform.openai.com/api-keys
   - Sign in / create account
   - Click "Create new secret key"
   - Copy the key

2. **In the App:**
   - Settings tab
   - Paste API key
   - Base URL: `https://api.openai.com/v1`
   - Model: `gpt-4o-mini` (cheaper) or `gpt-4o` (better)
   - Save Settings

**Cost:** ~$0.01-0.05 per resume  
**Speed:** ~1 minute per resume

#### Option B: Ollama (Local AI - Recommended for Privacy)

**New in v1.4.3!** Complete privacy - data never leaves your computer.

**One-Click Setup:**

1. **In the App:**
   - Settings tab → Scroll to "Local AI (Ollama)"
   - Click "Install Ollama" (~2 min)
   - Click "Download Recommended Models" (~7 GB, ~10-15 min)
   - Click "Test Local AI" to verify

2. **Done!** Ready for private resume generation

**Cost:** Free!  
**Speed:** ~2-3 minutes per resume

**Requirements:** 8GB+ RAM, 10GB disk space

---

### Step 3: Import Your Resume (1 min)

1. **Go to** Vault tab
2. **Click** "Import Resume"
3. **Select** your resume file (PDF, Word, or text)
4. **Wait** for extraction

**App creates your Master Profile automatically**

---

### Step 4: Add Job Description (1 min)

1. **Go to** Job Descriptions tab
2. **Option A:** Upload file (PDF/Word/text)
3. **Option B:** Paste text from website
4. **Click** "Analyze JD"

**App extracts requirements automatically**

---

### Step 5: Generate Resume (2 min)

1. **Go to** Generate tab
2. **Select** your profile
3. **Select** job description
4. **Choose** template:
   - **Modern** - Tech/startups
   - **Classic** - Finance/law
   - **Minimal** - Design/academia
   - **Executive** - Leadership
5. **Click** "Generate Resume"
6. **Wait** 30-60 seconds
7. **Review** the result
8. **Download** PDF

**Done! 🎉**

## 💡 Pro Tips

### Better Results

- ✅ **Select relevant experiences** for each job
- ✅ **Choose industry-appropriate template**
- ✅ **Review AI output** before submitting
- ✅ **Generate multiple versions** for A/B testing

### Save Money (OpenAI)

- Use `gpt-4o-mini` model (cheaper, still good)
- ~$0.01 per resume vs $0.05 for `gpt-4o`

### Faster Workflow

1. Import all your resumes at once (app merges them)
2. Save job descriptions for jobs you're interested in
3. Generate multiple resumes in one session

## ⚠️ Troubleshooting

### App Won't Open

**Right-click → Open** (bypasses Gatekeeper)

### Generation Fails

1. Check API key in Settings
2. Verify internet connection
3. Try different model (`gpt-4o-mini`)

### Can't Download PDF

Use browser's print → "Save as PDF"
